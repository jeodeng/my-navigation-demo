my-navigation-demo
==================

> A Vue.js project

一个demo，使用了vue-navigation，模拟了手机滑动的路由切换动画，并且记录滚动位置

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
